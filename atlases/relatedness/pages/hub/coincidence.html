Coincidence (Sandler interference map). Per-window-pair coefficient of coincidence C = r₁₂ / (r₁ · r₂), drawn as a 2D heatmap. C ≈ 1 = independence (no interference). C < 1 = positive interference (fewer DCOs than expected — the normal meiotic pattern). C ≫ 1 = negative interference (usually a sample-size or genotyping artefact). Off-diagonal cells inside an inversion span go to C → 0 or undefined: the loop disrupts the precondition / resolution chain, so the joint exchange rate collapses. Click any cell to read the pair detail in the inspector below. Karyotype matrix at the bottom.
NCO ≠ CO · WHAT C ACTUALLY MEASURES
Coincidence asks whether a crossover in interval A affects the chance of a crossover in interval B. The r₁ / r₂ / r₁₂ on this page are CO and DCO frequencies — chromosome-arm exchanges visible as long inherited-tract switches. Non-crossovers (NCO) are a different recombination outcome (short gene-conversion tracts, no arm exchange) and cannot be substituted into this formula. NCO traces the precondition / DSB initiation layer (see the Eligibility tab); CO traces the exchange outcome used here. With the current ngsPedigree / ngsTract data the safe estimability flags are: CO / inherited-tract switches: estimable; DCO / coincidence: partially estimable (power-limited); NCO: not estimable without much denser, phased variant data and a short-tract-aware caller; CO vs NCO resolution: not estimable from current data.
