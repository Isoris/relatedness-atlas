Batch cross planner. Multi-pair extension of the single-focal Compatibility tab. Generates every candidate parent pair under the chosen filters (sex-aware, exclude close kin, exclude same family hub, exclude carrier-of-FAIL inversions), ranks the pairs by a composite cross-utility score, and flags inheritance-side warnings on each (heterokaryote → CO suppression, shared-arrangement redundancy, family-only crosses). Read-only — no compute beyond aggregating the existing karyotype + family + sex inputs.

SCORING
Each pair receives a cross_utility_score ∈ [0, 1]:
  • +0.35 opposite-sex (when sex data exists for both)
  • +0.25 cross-family
  • +0.15 fraction of PASS candidates with differing karyotypes (recombination potential)
  • +0.15 1 − (top-hub-share confound) — averaged across each parent's heterokaryotypes
  • +0.10 fraction of PASS candidates where both parents are typed (coverage)
Then breeding-AI flags decorate the row (heterokaryote, redundancy, etc.) without changing the score.
