For one focal inversion, designs the marker panel that converts a candidate interchromosomal meiosis effect into a Pass-2 validation experiment (10k–50k catfish). Two marker layers: (1) focal-inversion-state markers — tag SNPs + breakpoint markers used to classify parents as het vs non-het; (2) transmission / recombination markers — ordered triplets m₁ → m₂ → m₃ on each tested chromosome so that two intervals A and B support CO, DCO, and coefficient-of-coincidence estimation in offspring panels. Per tested chromosome: READY / PARTIAL / LOW_POWER / MISSING. Export three TSVs (focal markers, interchromosomal test markers, test-design summary).

PAGE PURPOSE
This page converts a candidate interchromosomal meiosis effect into a practical marker panel: one set of markers to classify the focal inversion in parents, and another set of markers to detect transmitted CO/DCO patterns on other chromosomes in large offspring panels. Pass-1 (this atlas + WGS) discovers and ranks; Pass-2 (this panel) validates at scale.
